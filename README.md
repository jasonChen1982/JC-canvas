#### 为什么要做源码注释 ####

首先,自己对计算机图形学,webgl非常感兴趣,希望有时间可以学学.通过注释Three.js源码确实对学习计算机图形学,有非常大的帮助.

### 注释说明 ###
一开始打算直接在three.js文件中注释,但是由于github对超过1mb的文件无法直接在浏览器中查看的限制,而且为了方便以后的维护和管理,定改变注释的方式.以后将按照官方的目录结构,按单个文件方式进行注释.之前注释的代码在three.js文件中.
